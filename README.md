## Links

* [中文文档](https://github.com/Wscats/vue-awesome-mui)

## Install

```bash
npm install vue-awesome-mui -save
```

## Get Started

```javascript
import Mui from 'vue-awesome-mui'
import 'vue-awesome-mui/mui.css'
Vue.use(Mui)
```
